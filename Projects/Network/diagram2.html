Public and Private Network Architecture

Project Summary

This project involves creating a secure network architecture that integrates both public and private access points. This network setup allows a business to have a protected private network for its employees, while also giving public access WiFi for its guests/customers.

Conclusion

This project gave me a lot of insight into how private internet access and public wifi access are achievable in a small business network. This project also taught me how vulnerable public WiFi access really is and the security features needed to keep the network secure while also being efficient. By integrating public and private networks and implementing essential protocols, it is very possible to maintain a secure and efficient network environment.
