CyberApolis Water Breach Report

Chas Riley

University of Arizona

June 25, 2024

Project Overview

This project focuses on the authorized defensive penetration measures taken to defend the CyberApolis Municipal Water company from the terrorist group known as Carbon Spector. The report details the various phases of the penetration test, including reconnaissance, scanning, exploitation, post-exploitation, and mitigation measures.

Simulated Situation

The scenario involves a terrorist group taking control of the CyberApolis Municipal Water facility with the intent of flooding the city. The objective is to neutralize the threat by infiltrating the organizationâ€™s private server, gathering intelligence, and closing the dam floodgates to prevent disaster.

Objective

The objective of this penetration simulation is to assess the security posture of the CyberApolis Municipal Water company, identify vulnerabilities, and provide recommendations for mitigation. The report covers the following main topics:

  • Executive Summary: Summarizes the findings and key actions taken during the penetration test.
  • Introduction: Provides an overview of the incident and the immediate response by the security team.
  • Reconnaissance: Details the initial information gathering and identification of potential access points.
  • Scanning: Involves analyzing the website and network to identify vulnerabilities.
  • Exploitation: Describes the methods used to exploit identified vulnerabilities.
  • Post-Exploitation: Details the actions taken after gaining access to the system.
  • Summary and Mitigation: Summarizes the security setup and provides recommendations for improvement.
  • Synopsis: Offers a brief overview of the entire penetration test and its outcomes.
  • Appendix: Includes additional information and supporting documentation.

Tools Used

  • Burp Suite: A comprehensive tool for web application security testing, used for scanning and exploiting vulnerabilities.
  • Nmap: A network scanning tool used to discover hosts and services on a network.
  • John the Ripper: A password cracking tool used to recover passwords from password hashes.
  • Hydra: A tool for brute forcing login credentials on various protocols.
  • Wireshark: A network protocol analyzer used for capturing and analyzing network traffic.

Project Conclusion

The CyberApolis Water Breach Report provided valuable insights into the security vulnerabilities of the facility's systems. By following a structured penetration testing approach, critical flaws were identified and mitigated. The implementation of recommended security measures and regular assessments will significantly reduce the risk of future breaches.
